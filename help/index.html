Font icons

Icons

Symbolizes a command, action, or information using UI icons

Lamour has included FontAwesome Icons in its package and will update it regularly by new releases. So you're safe to use any icon from FontAwesome Four and FontAwesome Five (Free) , Flaticon websites.

Instruction for adding new icons form flaticon Library

To add new icons to our flaticon liberary check this link https://www.flaticon.com/collections/MjgyODgzNQ== .

This will redirect you to flaticon website where you can add all the icons to your collection using Add to my collection button. In order to do this you must logged in with flaticon. You can add more icons from Flaticon Library to your collection using search feature.

After compleating your collection click on Download Collection and download iconfont format which you can use into site and customize it fully.

Now we need to update our existing flaticon.css file for add new icons so first please take a backup of css folder.

Now we need to replace new fonts and css files with existing one. So go ahead and navigate to font directory and copy Flaticon.eot, Flaticon.svg, Flaticon.ttf, Flaticon.woff files and replace this into your template's Fonts directory.

Second step is to replace flaticon.css file to your template's css directory. After replacing both Fonts and css files. We need to change a little code for flaticon.css.

Open Flaticon.css in any text-editor and replace the following lines. 


/*
Flaticon icon font: Flaticon
Creation date: 21/05/2018 14:11
*/

@font-face {
  font-family: "Flaticon";
  src: url("../fonts/Flaticon.eot");
  src: url("..fonts/Flaticon.eot?#iefix") format("embedded-opentype"),
       url("../fonts/Flaticon.woff") format("woff"),
       url("..fonts/Flaticon.ttf") format("truetype"),
       url("../fonts/Flaticon.svg#Flaticon") format("svg");
  font-weight: normal;
  font-style: normal;
}

@media screen and (-webkit-min-device-pixel-ratio:0) {
  @font-face {
    font-family: "Flaticon";
    src: url("../fonts/Flaticon.svg#Flaticon") format("svg");
  }
}

Now your new added icons are ready to use you can get new font class names by opening Flaticon.html file from your font folder of flaticon collection.
